Painting Program

In this project, you will write a simple painting program.

Getting Mouse Position

We need to know where the mouse is, before we can paint there. We can ask for a particular function to be called whenever the mouse moves, the same way we did for buttons and keys on the keyboard. The browser will call our function with a dictionary that describes where the mouse is. There's a catch, though. pageX and pageY give us coördinates relative to the upper left corner of the web page. To draw in an SVG, we need coördinates relative to the corner of the SVG! So we also need to find where the SVG is on the page.

When you move your mouse on the SVG below, the text updates to show the two positions. Read the Javascript code to see how it works.

Using Mouse Position

Make a new SVG with a small circle in it. Write JS code that reads the mouse position, and moves the circle to follow the mouse.

Append to SVG

To append a new circle or other shape to an SVG, we need to write slightly different code than appending an li to a ul. Use the function called svg to make the circle, and attr to set its attributes. Then wrap the new element in append as usual. Read the code that draws the shapes below, for more details.

Painting

Copy your code from above. Instead of moving the circle, add a new circle to the image each time the mouse moves.

Color, Brush, Opacity

Add at least two of the following features, to make your paint program more useful:

  • a text field to set the brush size (radius of the circle)
  • a text field to set the fill-opacity property
  • a text field to set the color, using #13ae45 syntax
  • a pencil tool that draws lines between mouse positions, instead of circles
