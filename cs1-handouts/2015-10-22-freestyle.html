Instead of reproducing a page designed by someone else, design your own page. Some ideas:

  • A choose-your-own adventure story, with multiple pages and links between pages.
  • A blog with a few posts, and the same HTML structure repeated for each.
  • A personal homepage with a portfolio of your best (art?) work from the last few years.

Whatever the topic, the point is to show the HTML and CSS skills that you have learned so far this year. Make your website as great as you know how. The following are required to get an A:

  • At least 8 different HTML elements within the body of the document, used appropriately. (<html>, <head>, <link> and the rest are not included in these 8.)
  • A link to another page, either one you write or elsewhere on the web
  • 3 elements that share the same class
  • At least 12 CSS selectors total
  • At least one CSS selector by element name, one by ID, and one by class name
  • At least 8 different CSS properties (eg, color, float, width, margin)
  • At least one CSS media query, to set different CSS at different sizes
  • Appropriate accomodations for at least one type of disability. It can be the one that you're researching, or any other researched by someone in class.

Save your project in the same folder with all your other projects, in a folder starting "08_"
