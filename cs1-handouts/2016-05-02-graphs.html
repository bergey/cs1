First Data Analysis Project

  1. Load the D3 library with a script tag pointing to https://cdnjs.cloudflare.com/ajax/libs/d3/3.5.16/d3.js
  2. Pick a data set from the data folder. In a subsequent project you will have more scope to pick your own data.
  3. Make a scatterplot comparing two variables.
    • Decide which columns of data to plot on the X & Y axes
    • Decide whether to plot all rows or only some. For example, PA only, or major cities only, or just one year or month
    • Use d3.csv or d3.tsv to load the data
    • Use d3.scale to fit the data to the available pixels (X and Y)
    • Use d3.svg.axis to add axis ticks and labels
    • Label each axis with a description and the units
  4. Make a second plot, of a different type. Choose a plot type that makes sense for your data, using what you learned from the first plot. If you're not sure what to plot, talk to me.
