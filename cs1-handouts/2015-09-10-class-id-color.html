Relating HTML to CSS

CSS Selectors

element name
Any element name can be a CSS selector. This applies to all emphasized text: em { color: red; }
class
Any HTML element can have a class, and any number of elements can have the same class. In HTML, we add a class called "important" by writing <em class="important"> We can refer to that in CSS by writing .important {/*properties here*/}
id
Any HTML element can have an ID attribute. It's similar to a class, but only one element in a document can have a given ID. If you're not sure which to use, use a class. When we use Javascript, we'll see reasons to prefer an ID. We refer to an ID in CSS with the # character, like #myIdentity {/*properties here*/}

HTML Links

The a element, short for anchor, is used to make a link. The URL that is the destination of the link is given as the href attribute. The link text goes inside the a element. You can also have links that are images or divs, instead of text. An example: <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a">Click Me!</a>

Color

Often we want to specify a color in our CSS. CSS defines a bunch of common color names. When you want a less common color, you can specify it as the red, green, and blue parts, in that order. For example, rgb(80%,0%,80%) is a shade of purple. There are a few other ways, also, which you may see reading others' code. Without the % signs, each color component is a number between 0 and 255 - dividing that number by 255 is the same as the percentage value. Or you may see #CC00CC, in which case each 2 numbers is one color component, between 0 and 255, in base 16. As usual, you can read more on MDN.

Computer screens produce color by mixing different proportions of red, green, and blue light. They can't reproduce every color that occurs in the world (or in print), but this works pretty well, because the human eye also has 3 kinds of cell to detect color. Computer monitors vary quite a bit in what color they produce for a given set of numbers - so pick a color you like, but try not to agonize too much over small differences.
