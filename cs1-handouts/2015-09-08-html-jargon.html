Now that you've spent some time reading & writing HTML, here are some terms used to refer to particular parts of HTML. These should make it easier to understand other writing about HTML.

Start Tag
An element name enclosed in angle brackets, like <ol>, with optional attributes.
End Tag
An element name enclosed in angle brackets and slash, like </ol>. Most elements have a start tag and an end tag, but a few have only a start tag. May also be called a closing tag.
Element Name
One of the elements defined by HTML, like p, h1, or li.
Attribute
Additional information included as part of a start tag. For example, the src part of <img src="images/something.jpg">. Every attribute has a key before the equals sign, and a value after the equals sign. The value must be inside quotation marks.
Element
An HTML element is the start tag, the end tag, and all the HTML in between. Element can also refer to the rendered equivalent in the browser.
Parent
The parent or parent element of a particular element is the element that contains it. For example, in <ul><li>one</li></ul>, ul is the parent of li.
Child

A child or child element of a given element is any element that has it as a a parent. The ul in this example has two children, both li.

<ul>
  <li>one</li>
  <li>two</li>
</ul>
Empty Element
An empty element is one that is never allowed to have children. These should always be written without a end tag. The most common example is img. You can see The full list on MDN.
