1. Sign up for a new account on github.com

    I suggest using your friendscentral.org email address.

  2. Make a copy of our next project

    Go to my github, at github.com/cs1_19_dictionaries. Click the button labeled Fork in the upper righthand corner. To "fork" means to make a copy where you intend to make your own changes.

  3. Sign up for Cloud 9

    Go to c9.io. Click the Octocat icon in the upper right, to sign in using your Github account.

  4. Open the Dictionaries project in Cloud 9

    Click the "Repositories" link on the left hand side. You'll see a list of your Github projects, which right now is just cs1_19_dictionaries. Click clone to edit next to this project. Cloud 9 will open a code editor. Right click on dictionaries.html and select preview to see the rendered website.

  6. Start writing functions in student.js
